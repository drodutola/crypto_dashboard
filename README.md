# Crypto Futures Market Dashboard 📊

A Python-based real-time dashboard for monitoring cryptocurrency futures markets — aggregating live price data, open interest, funding rates, and market sentiment into a single visual interface.

## Overview

Futures market analysis requires tracking multiple data streams simultaneously. This dashboard consolidates key metrics from crypto futures exchanges into a clean, real-time interface to support informed trading decisions.

## Features

- Live futures price feeds and price change tracking
- Open interest and volume monitoring
- Funding rate display across major pairs
- Market sentiment indicators
- Multi-asset support (BTC, ETH, and altcoin futures)

## Tech Stack

- **Language:** Python
- **Libraries:** Pandas, Matplotlib / Plotly, Requests
- **Data Sources:** CoinGecko API, CME Group API, exchange REST APIs

## How to Run

```bash
git clone https://github.com/drodutola/crypto_dashboard
cd crypto_dashboard
pip install -r requirements.txt
python dashboard.py
```
